Guidelines

Color

La paleta de color refleja un estilo y apariencia unificado y reconocible, el color primario y secundario generalmente refleja la marca y sirven para acentuar elementos. Variaciones claras o oscuras se pueden aplicar a cada color para diferentes elementos de la interfaz.
